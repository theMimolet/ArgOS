# ArgOS

## What is ArgOS ?

ArgOS is system image which sole purpose is to add MS Surface support on top of Bluefin.

It will remain maintained as long as a friend of mine still has his Surface Device.

## Hardware Support

It will target Surface Pro 4 devices in priority.

## What is a Bluefin ? Can I eat it ?

Haha, no, Bluefin is a Gnome-focused system image based on Fedora.

You can check them out [on their website](https://projectbluefin.io/)
